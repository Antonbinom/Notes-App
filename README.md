# Note-App

## Описание

Приложение планировщик/ежедневник

## Функционал

- Добавление/удаление заметок
- Добавление тегов для заметок
- Сохранение заметок при перезагрузке


## Стек технологий

- Vue3
- Vuex - хранилище состояний
- Options Api
- Vue-router - переход на страницы без перезагрузки
- Vue-cli
